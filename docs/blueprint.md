# **App Name**: DroughtCast

## Core Features:

- Data Ingestion: Collect and pre-process historical climate data from various sources including rainfall, temperature and El Nino indices.
- Predictive Modeling: Use an AI model tool to analyze historical climate data and predict future drought likelihood and severity. 
- Risk Assessment Map: Display a map visualizing drought risk levels across different geographic regions, incorporating predictive model outputs.
- Severity Scale: Provide a simple scale/legend to inform on the drought severity based on the projected weather forecast.
- Risk Alert: Provide drought alerts in UI for high-risk areas.

## Style Guidelines:

- Primary color: Sandy Brown (#F4A460), evoking dry earth and landscapes.
- Background color: Light Beige (#F5F5DC), creating a neutral and arid atmosphere.
- Accent color: Pale Goldenrod (#EEE8AA), for alerts, to suggest sunlight.
- Body and headline font: 'PT Sans', a modern sans-serif with a bit of warmth.
- Use weather-themed icons. Consider icons showing water levels, rainclouds and drought indicators.
- Implement a map-centric layout, ensuring intuitive navigation, making the affected zones as obvious as possible
- Use subtle animations for map zoom and drought progression. Subtle animations should also present/dismiss drought alerts.
